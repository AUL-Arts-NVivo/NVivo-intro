# NVivo-intro
Velkommen til vores NVivo kursus.

NVivo er et digitalt analyseværktøj, der er god til at strukturere dine data, kode for mening og til at give indsigter på tværs af materialet. Det bliver brugt indenfor mange forsjellige forskermiljøer, især indenfor humaniora og socialvidenskaberne, hvor man arbejder med kvalitative data. Aarhus Universitet giver mulighed for at købe en billig licens til programmet, og Aarhus uÚniversitetbibliotek underviser i og supporterer  programmet.
